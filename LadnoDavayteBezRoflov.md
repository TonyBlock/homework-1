## Страница входа

https://brrrello.ru/login

  - Поле "Логин"
    - по клику на инпут текст "Логин" уезжает над инпут;
    - по клику на инпут заполняется полоска инпута;
    - поле автозаполняется сохраненным логином при его наличии;

    - Цветовая валидация инпута ок:
      - Длина: валидно с 3 по 20 включительно, остальное - невалидно;
      - Содержание: латиница и цифры валидны, кириллица и специальные символы невалидны;
    
    - По снятию фокуса с инпута при введенном логине текст "Логин" не возвращается в инпут;
    - По снятию фокуса с инпута при введенном логине полоска не сбрасывается в исходное состояние;

  - Поле "Пароль"
    - по клику на инпут текст "Пароль" уезжает над инпут;
    - по клику на инпут заполняется полоска инпута;
    - поле автозаполняется сохраненным паролем при его наличии;

    - Цветовая валидация инпута не ок:
      - **Баг** Длина: невалидно до 6 символов включительно, дальше валидно;
      - ***Ожидаемое поведение*** Длина: валидно с 6 до 25 включительно, остальное невалидно;
      - Содержание: пароль, состоящий из латиницы, валиден;
      - Содержание: пароль, начинающийся с цифры, невалиден;
      - Содержание: пароль, состоящий из латиницы с цифрами не на первой позиции, валиден;
      - Содержание: пароль, состоящий из кириллицы, невалиден;
    
    - По снятию фокуса с инпута при введенном пароле текст "Пароль" не возвращается в инпут;
    - По снятию фокуса с инпута при введенном пароле полоска не сбрасывается в исходное состояние;

  - Ссылка "Создать аккаунт" ок:
    - корректно ведет на https://brrrello.ru/register;

  - По клику на кнопку "Войти" с корректной парой логин/пароль происходит редирект на https://brrrello.ru/boards;

  - По клику на кнопку "Войти" с некорректной парой логин/пароль редиректа не происходит:
    - На странице показываются два сообщения об ошибке с текстом "Неверный логин или пароль";

  - По клику на кнопку "Войти" с незаполненными инпутами показываются два сообщения об ошибке:
    - Под инпутом "Логин": "Введите логин длиной от 3 до 20 символов";
    - Под инпутом "Пароль": "Введите пароль длиной от 6 до 25 символов";

  - По клику на кнопку "Войти" с незаполненным инпутом "Логин" показывается одно сообщение об ошибке:
    - Под инпутом "Логин": "Введите логин длиной от 3 до 20 символов";
    - Введенное пользователем значение инпута "Пароль" сохраняется;

  - По клику на кнопку "Войти" с незаполненным инпутом "Пароль" показывается одно сообщение об ошибке:
    - Под инпутом "Пароль": "Введите пароль длиной от 6 до 25 символов";
    - Введенное пользователем значение инпута "Логин" сохраняется;

## Страница со списком команд

https://brrrello.ru/boards

  - "Личное пространство" не ок: 
    - Всегда отображается первой в списке команд;
    - У пространства отсутствует кнопка "Удалить";
    - У пространства отсутствует кнопка "Редактировать";
    - У пространства отсутствует счетчик участников и кнопка "Добавить участника";
    - У пространства отображается кнопка добавления доски:
      - Кнопка добавления доски ведет на модальное окно создания доски со значением "Личное пространство" в поле "Команда";
    - В пустом личном пространстве отображается текст "Похоже досок пока нет. Создайте Ваш первый проект в личном пространстве.";
      - Ссылка "Создайте" ведет на модальное окно создания доски со значением "Личное пространство" в поле "Команда"
    - **Баг** При смене логина пользователя не меняется название личного пространства;
    - ***Ожидаемое поведение*** При смене логина пользователя название личного пространства обновляется с учетом нового логина;

  - Команда ок:
    - Старые команды отображаются в алфавитном порядке;
    - Новые команды добавляются снизу и при перезагрузке перерисовываются в алфавитном порядке;
    - В пустой команде отображается текст "Похоже досок пока нет. Создайте первый проект команды "..."":
      - Ссылка "Создайте" ведет на модальное окно создания доски со значением названия текущей команды в поле "Команда";
    - У команды отображается кнопка "Удалить":
      - При клике на кнопку отображается модальное окно удаления текущей команды;
    - У команды отображается кнопка "Редактировать":
      - При клике на кнопку отображается модальное окно редактирования команды со значением названия текущей команды в поле "Название команды";
    - У команды отображается кнопка добавления доски:
      - Кнопка добавления доски ведет на модальное окно создания доски со значением названия текущей команды в поле "Команда";
    - У команды отображается кнопка "Добавить участника" и счетчик участников команды:
      - При клике на кнопку отображается модальное окно добавления участника в текущую команду;

  - "Остальные доски" ок:
    - Отображается только в том случае, если есть доски, в которые приглашен пользователь и при этом пользователь не является участником команды, создавшей доску;
    - Отображаются последней в списке команд;
    - У раздела отсутствует кнопка "Удалить";
    - У раздела отсутствует кнопка "Редактировать";
    - У раздела отсутствует счетчик участников и кнопка "Добавить участника";
    - При уходе пользователя из доски доска исчезает из раздела;

### Модальное окно создания команды

 - Модальное окно создания команды ок:
  - По клику на крестик модальное окно закрывается, не создавая команду;
  - По клику на кнопку "Создать" с пустым инпутом "Название команды" отображается ошибка с текстом "Название команды слишком короткое";
  - По клику на кнопку "Создать" с любым текстом в инпуте "Название команды" модальное окно закрывается, создавая команду с введенным названием;

### Модальное окно удаления команды

 - Модальное окно удаления команды ок:
  - По клику на крестик модальное окно закрывается, не удаляя команду;
  - По клику на кнопку "Нет, отменить" модальное окно закрывается, не удаляя команду;
  - По клику на кнопку "Да, удалить" модальное окно закрывается, удаляя команду;

### Модальное окно редактирования команды

 - Модальное окно создания команды не ок:
    - **Баг** Отображается текст "Создание команды";
    - ***Ожидаемое поведение*** Отображается текст "Редактирование команды";
  - По клику на крестик модальное окно закрывается, не изменяя команду;
  - По клику на кнопку "Сохранить" с пустым инпутом "Название команды" отображается ошибка с текстом "Название команды слишком короткое";
  - По клику на кнопку "Сохранить" с любым текстом в инпуте "Название команды" модальное окно закрывается, изменяя название команды на введенное;

### Модальное окно добавления участника в команду

 - Модальное окно добавления участника в команду ок:
  - В списке участников отображаются все участники команды;
  - По клику на участника он исключается из команды;
  - По вводу трех символов в инпут происходит поиск участника. По добавлению символов поисковые результаты корректируются;
  - По клику на участника на поиске иконка плюса заменяется на иконку галочки и участник добавляется в команду;
  - По клику на крестик модальное окно закрывается, не изменяя список участников;

### Модальное окно создания доски 

 - Модальное окно создания доски ок:
  - По клику на крестик модальное окно закрывается, не создавая доску;
  - По клику на кнопку "Создать" с пустым инпутом "Название доски" отображается ошибка с текстом "Название доски слишком короткое";
  - В выпадающем меню "Команда" отображается список команд пользователя;
  - По клику на кнопку "Создать" с любым текстом в инпуте "Название доски" модальное окно закрывается, создавая доску в соответствующей команде;
